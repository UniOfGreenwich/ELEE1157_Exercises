Exercise 1: Identify the IPv4 Class

  1. Question: What class does the IP address 126.25.34.2 belong to?

    Answer
    • Class: Class A

  2. Question: What class does the IP address 191.50.23.14 belong to?

    Answer
    • Class: Class B

  3. Question: What class does 224.0.0.5 belong to, and what is its purpose?

    Answer
    • Class: Class D
    • Purpose: Class D addresses are used for multicast.

Exercise 2: Calculate the Number of Hosts and Networks

  1. Question: For a Class A network, how many hosts and networks are possible?

    • Hint: Class A networks use the first 8 bits for the network, leaving 24 bits for hosts.

      Answer
      • Networks: \(126 (1.0.0.0 - 126.0.0.0)\)
      • Hosts per network: \(2^{24} - 2 = 16,777,214\) usable hosts

  2. Question: Calculate the number of hosts and networks for a Class C address.

    • Hint: Class C uses 24 bits for the network and 8 bits for hosts.

      Answer
      • Networks: \(2^{21} = 2,097,152\) possible Class C networks
      • Hosts per network: \(2^{8} - 2 = 254\) usable hosts

Exercise 3: Convert Between Binary and Dotted Decimal Notation

  1. Question: Convert the binary IP 11000000.10101000.00000001.00000001 to dotted decimal notation.

    • Hint: Break down each octet and convert each binary section to decimal.

      Answer
      • Dotted Decimal: 192.168.1.1

  2. Question: Convert the IP address 10.0.0.1 to binary.

    • Hint: Convert each octet (10, 0, 0, and 1) individually to binary.

      Answer
      • Binary: 00001010.00000000.00000000.00000001

Exercise 4: Reserved and Special Addresses

  1. Question: Is 127.0.0.1 a public or reserved address, and what is it used for?

    • Hint: Think of testing or diagnosing a local machine.

      Answer
      • Type: Reserved (Loopback address)
      • Purpose: Used for local testing of the TCP/IP stack.

  2. Question: Identify the purpose of the IP address range 169.254.0.0 - 169.254.255.255.

    • Hint: This range is automatically assigned when DHCP fails.

      Answer
      • Type: Automatic Private IP Addressing (APIPA)
      • Purpose: Assigned when DHCP is unavailable, allowing local communication.
